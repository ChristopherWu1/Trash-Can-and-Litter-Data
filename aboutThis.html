About The Project

This project was made by Christopher Wu for CSci 39542: Introduction to Data Science at Hunter College. I used python and a number of libraries to import the data, manipulate it, and visualize it using graphs and maps.

List of libraries

  • Pandas
  • Numpy
  • Seaborn
  • matplotlib.pyplot
  • folium
  • folium MarkerClusters
